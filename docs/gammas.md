# Detect gamma from note set
```bash
$ harmony_tools gammas detect A B D_sharp
C_sharp_minor   E  F# G# A  B  C# D#
E_major         E  F# G# A  B  C# D#
```


# Show gamma
```bash
$ harmony_tools gammas show A_major B_major
A_major         A  B  C# D  E  F# G#
B_major         B  C# D# E  F# G# A#
```
